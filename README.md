# vue-demo
使用VUE2.x + vuex + vue-router + element ui + socket io +echars,做了点简单的页面.
src/server文件夹里放了socket server的代码，需要单独npm i 和 node server 运行，监听端口4000
其中一个页面的数据是通过socket push过去的

## Project setup
```
npm install
```
